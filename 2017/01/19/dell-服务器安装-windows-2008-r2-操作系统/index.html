Dell-服务器安装-Windows-2008-R2-操作系统

发表于2017-01-19|更新于2026-06-11|技术

前言

这是Dell服务器装机教程的第2篇，全系列包括：

  1. Dell服务器做磁盘阵列
  2. Dell 服务器安装 Windows 2008 R2 操作系统
  3. Dell服务器简单配置

系统安装比较简单，和普通安装不同的是需要安装磁盘驱动，否则安装时会提示没有可用的磁盘。需要提前准备好服务器的驱动程序。

步骤：

  1. 上一篇教程已经设置从光盘启动，服务器启动时会提示 “Press any key to boot from CD.”，只需按任意键即可加载系统安装光盘。

  2. 我们选择安装企业完整版。其他安装根据具体情况选择。

  3. 一路确认进入磁盘选择页面，没有可用磁盘，则选择加载驱动驱动。

  4. 清除旧的磁盘。当驱动正常加载后，点击“驱动器选项（高级）”，先将旧的磁盘删除。

  5. 选择安装盘。选中已经分好的80GB磁盘，点击“新建”，然后点击“下一步”开始安装系统。
