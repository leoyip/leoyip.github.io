Dell服务器简单配置

发表于2017-01-21|更新于2026-06-11|技术

前言

这是Dell服务器装机教程的第3篇，全系列包括：

  1. Dell服务器做磁盘阵列
  2. Dell 服务器安装 Windows 2008 R2 操作系统
  3. Dell服务器简单配置

Dell服务器简单配置包括以下内容：

  • 检查磁盘分区情况
  • 安装必要驱动程序
  • 安装远程控制软件Radmin
  • 激活 Windows 2008 R2

检查磁盘分区情况

服务器安装完系统之后，应当及时检查磁盘划分情况。有时候会遇到磁盘2TB大小限制的情况，这个问题的解决会在其他教程中详细讲解。

查看磁盘划分的步骤：

  1. 点击windows按钮旁边的工具箱，在计算机管理页面左侧导航栏找到磁盘管理选项。
  2. 打开磁盘管理器，即可查看磁盘划分情况。
  3. 确认无误后关闭窗口。

安装必要驱动程序

Dell服务器安装系统之后，必须安装芯片组驱动、网卡驱动以及光纤卡驱动才可以正常使用。相应的驱动程序由供应商提供。提前准备好保存在U盘文件内。

安装芯片组驱动

先安装 .Net Framework 3.5.1

  1. 在开机显示的的角色与功能管理器中下拉，找到自定此服务器>添加功能。

  2. 在选择功能页面，点击 “.NET Framework 3.5.1 功能”
    前的 “+”号，在展开列表中勾选“.NET Framework 3.5.1 ”。然后点击下一步，等待安装完毕即可。

安装芯片组驱动软件

打开U盘，找到提前准备的驱动程序，点击安装Chipset_Driver_xxx.exe,按照默认设置安装即可。
**注意：**在安装芯片组驱动程序时，不要进行其他安装，芯片组驱动程序安装完毕立即重启服务器。

安装网卡驱动

  1. 在工具文件夹下，打开**（3）Network文件夹，打开子文件夹Broadcom NetXtreme Network Drive vxx.x.x**文件夹。
  2. 使用默认设置点击运行“Network_Driver_xxx.exe”即可。

安装光纤驱动

  1. 在工具文件夹下，打开**（3）Network文件夹，打开子文件夹Qlogic BCM57xx and BCM57xxx NetWork Driver vxx.x.x**。
  2. 使用默认设置安装驱动程序即可。

网络测试

  1. 先进入： 控制面板>网络和Inernet>网络共享中心，点击左侧导航栏中的“更改适配器设置”。
  2. 选择一个网络连接来设置IP，具体网口及IP以实际需要来设定。IP设置步骤略。
  3. IP设置好了需要测试能否ping通网关，ping测试步骤略。

安装远程控制软件Radmin

简介

Radmin 是一功能较多的安全远程控制软件，可以从多个地点访问同一台电脑，并使用高级文件传输、远程关机、Telnet、操作系统集成的 NT 安全性系统支持，以及其它功能。

安装配置

  1. 将工具包文件夹目录下的Radmin.rar文件解压到“当前文件夹”。
  2. 在解压出来的Radmin34文件夹内点击运行rsevr34cn.exe，按默认参数安装即可。
  3. 安装完毕后，默认开始Radmin服务器设置。点击“选择”按钮，不勾选使用缺省的端口，具体端口根据需求设定，然后点击确定完成设置。
  4. 返回Radmin服务器设置窗口，点击“使用权限···”，新建管理员帐号，设置用户名及密码。下方权限勾选全部存取。

激活Radmin Server

  1. 在Radmin34文件夹内找到NewTrialStop23.rar，解压到“NewTrialStop23\”文件夹内。
  2. 然后再在“NewTrialStop23\”文件夹内，以管理员权限运行install.exe。
  3. 在弹出的命令行窗口中，按照提示操作完成激活。

激活 Windows 2008 R2

我们使用windows loader激活系统，相应程序保存在工具包文件夹下面。
步骤：

  1. 在工具包文件夹下找到windowsloader_x.x.x文件夹，运行该文件夹内的“Windows Loader.exe”文件，在弹出面板中选择“install”。

  2. 系统激活后，需要重启服务器。

到此Dell服务器的基本配置完成了，可以开始在上面部署相应的应用。
