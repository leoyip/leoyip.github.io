#001 第一次开车上高速

发表于2018-05-02|更新于2026-06-11|1000个生活片段

这次，漫长的一个小时车程里，好在有惊无险，这是二十六岁的幸运，也是十八岁的幸运。

昨天，那个教我骑自行车的男人，陪同我第一次开车上高速公路。和那时骑自行车不一样，这次老妈和弟弟也在车上。一家人陪同我从珠江东边去到珠江西边，全程47.7公里，耗时1小时12分钟。然而，这却是有生以来感觉最漫长的一个小时。

我坐上驾驶座，生疏地调整座位、扣上安全带，笨拙的启动汽车。老爸的车已有一些历史，它至少八年前就已经加入我们家了。

八年前，一家人在东莞过年。大年初六，父亲坐在驾驶位上，我在副驾驶上，长途驱车回粤西老家。在市里的一段路上，父亲刹车不及，撞上一位横穿马路的老人。老人连带他的自行车，重重的砸在挡风玻璃上，然后弹出几米外。当我从副驾驶走下来时，老人倒在血泊中，柔软的四肢扭曲变形·····

车祸之后，家里发生不少变故，八年时间重重碾过。现在，父亲坐到副驾驶上，我坐在驾驶位上，还是这辆车。

方向盘有偏左一些角度，不像正常的汽车可以平稳握持，这让我路上行驶的时候行车容易蛇行前进。档位也挺沉重，每次切换档位都要花很大力气，后面再高速路上还出现了挂不上五档的小险情。老车有它的毛病，新手上路的我也有各种毛病：过红绿灯启动不自然，甚至还出现了一次死火；忘记打转向灯。最危险的是一紧张忘记在高速路上怎样用点刹来制动。

如果不是老爸在副驾驶上不停地提醒、警告，恐怕我这个马路杀手早就报销驾照了。可能在我内心深处，还是那一位十八岁刚经历车祸的少年，一脸木然不知所措地站立在那里。父母说给我壮胆，像老鹰推着小鹰离开悬崖石壁上的窝巢。这次，漫长的一个小时车程里，好在有惊无险，这是二十六岁的幸运，也是十八岁的幸运。

在这一小时里，尽管一直紧绷着神经，但是追逐着夕阳前行，景色相当壮丽。从来没有想过，驾驶位上看到高速穿越落霞的广阔时空是这么震撼。当时想着，如果汽车能够自动拍照就好了。就这么一会出神，差点发生追尾事故。
