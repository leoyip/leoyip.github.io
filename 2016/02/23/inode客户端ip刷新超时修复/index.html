前言

从前天一早开始，校园网就断了，各种报错，具体原因现在还没有搞清楚。

开始时报

然后是

总之上不了网。

修复过程

  1. 前天以为是断电原因，没有修复。
  2. 更换iNodeMon.exe和WlanTest.exe文件再修复客户端，无效。
  3. 禁用虚拟机网卡和wifi网卡，无效。
  4. 重装客户端，报错变为
  1. 上网查看帐号余额，为零，问题是用同学的电脑可以连上，说明校园网可用。
  2. 参考网上帖子卸载了网卡驱动，然后再连接，发现有两个网卡，从上到下一一使用，第二张网卡可用，终于连上校园网。


结语
