# 前言
这是Dell服务器装机教程的第2篇，全系列包括：

1. [Dell服务器做磁盘阵列](http://www.jianshu.com/p/59826ffecbaf)
2. [Dell 服务器安装 Windows 2008 R2 操作系统](http://www.jianshu.com/p/0f567a0126bf)
3. [Dell服务器简单配置](http://www.jianshu.com/p/ea9ec683d319)

系统安装比较简单，和普通安装不同的是需要安装磁盘驱动，否则安装时会提示没有可用的磁盘。需要提前准备好服务器的驱动程序。
# 步骤：	
	
1. 上一篇教程已经设置从光盘启动，服务器启动时会提示 “Press any key to boot from CD.”，只需按任意键即可加载系统安装光盘。
![](/images/imported/upload-images-jianshu-io-upload-images-447752-9a9d0884046a890d-jpg.jpg)

2. 我们选择安装企业完整版。其他安装根据具体情况选择。
![](/images/imported/upload-images-jianshu-io-upload-images-447752-cfa5c88472ca94bb-jpg.jpg)

3. 一路确认进入磁盘选择页面，没有可用磁盘，则选择加载驱动驱动。
![无可用磁盘](/images/imported/upload-images-jianshu-io-upload-images-447752-015f95cef964e235-jpg.jpg)
![浏览加载驱动程序](/images/imported/upload-images-jianshu-io-upload-images-447752-5a0de4c228c04649-jpg.jpg)

4.  清除旧的磁盘。当驱动正常加载后，点击“驱动器选项（高级）”，先将旧的磁盘删除。
5.  选择安装盘。选中已经分好的80GB磁盘，点击“新建”，然后点击“下一步”开始安装系统。
