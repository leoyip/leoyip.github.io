Moo.do和workflowy是谷歌浏览器上面两款著名的GTD工具，前者功能大而全，后者专注简约，两者的工作语境有很大的不同。面对多任务的环境时，适合使用Moo.do。个人使用一段时间后，发现Moo.do的快捷键纷繁复杂，有时多个快捷键对应一个功能，不便记忆。于是，我作了一些精简，把个人常用的功能简记如下。仅供参考。

### 1. 基本
|快捷键|功能|
|---|---|
|Tab|缩进一个item|
|shift + Tab|缩退一个item|
|Ctrl + /|完成一个项目|
|Ctrl + 1、2、3|设置优先级|
|Ctrl +8|开始一个项目|
|Alt + Up| 收起项目|
|Alt + Down|展开项目|
|Ctrl + F |检索文字|

### 2. 导航
|快捷键|功能|
|---|---|
|Ctrl + Enter	|展开项目页|
|Ctrl + Shift + Enter |退出项目页|
|Ctrl + Home	|跳转到第一个项目|
|Ctrl + End |跳转到最后一个项目|

### 3. 高级
|快捷键|功能|
|---|---|
|Ctrl + Shift + / |将已完成的项目归档隐藏|
|Ctrl + Shift + Up	|将项目上移|
|Ctrl + Shift + Down |将项目下移|
|Ctrl + Shift + .| 展开或收起顶级列表|

### 4. 文本编辑器
|快捷键|功能|
|---|---|
|Ctrl + Z |撤销上一操作|
|Ctrl + Y |重做上一操作|
|Ctrl + Backspace |删除前一个单词|
|Ctrl + Delete	|删除后一个单词|
