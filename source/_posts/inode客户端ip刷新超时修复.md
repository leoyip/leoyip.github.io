### 前言
从前天一早开始，校园网就断了，各种报错，具体原因现在还没有搞清楚。

开始时报
```
IP刷新超时
```

然后是
```
2016-02-23 10:58:12 网络连接中断 ---- 没有找到匹配的网卡，网卡可能被禁用、卸载或重新安装。请确认网卡的状态或重新选择认证网卡。
2016-02-23 10:58:24 连接已断开`

```
总之上不了网。

### 修复过程
1. 前天以为是断电原因，没有修复。
2. 更换*iNodeMon.exe*和*WlanTest.exe*文件再修复客户端，无效。
3. 禁用*虚拟机网卡*和*wifi*网卡，无效。
4. 重装客户端，报错变为
```
2016-02-23 10:58:12 网络连接中断 ---- 没有找到匹配的网卡，网卡可能被禁用、卸载或重新安装。请确认网卡的状态或重新选择认证网卡。
2016-02-23 10:58:24 连接已断开
```
5. 上网查看帐号余额，为零，问题是用同学的电脑可以连上，说明校园网可用。
6. 参考网上帖子卸载了网卡驱动，然后再连接，发现有两个网卡，从上到下一一使用，第二张网卡可用，终于连上校园网。

### 结语
除了浪费时间，没什么好说的了。只能将故障和解决方法记录下来，以后再遇到类似问题，可以有得借鉴。
